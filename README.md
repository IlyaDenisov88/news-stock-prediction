# news-stock-prediction

Дипломный проект: анализ влияния новостных данных на динамику цен акций российского фондового рынка с применением методов машинного обучения.

## Основные этапы
- сбор новостей
- сбор котировок MOEX
- предобработка и merge
- генерация признаков
- обучение моделей
- анализ результатов

## Чтобы код успешно запустился локально, необходимо установить зависимости:
`pip install -r requirements.txt`
